# Jak zgłaszać błędy

Znalazłeś błąd w usłudze White Ravens? Świetnie, że chcesz go zgłosić! Dzięki temu pomagasz nam rozwijać nasze projekty. Błędy zgłaszamy na GitHub lub na Discordzie.

!!! tip "Repozytoria White Ravens"
    [GitHub White Ravens](https://github.com/orgs/whiteravens20/repositories)

---

## Szybka ścieżka: zgłoś na Discordzie

Nie masz konta GitHub lub nie wiesz, gdzie zgłosić? Napisz po prostu na naszym [serwerze Discord](https://discord.gg/5JMk8Z4) — pomożemy.

---

## Zgłaszanie na GitHub — krok po kroku

### Krok 1: Załóż konto

Jeśli nie masz konta na GitHub, wejdź na [github.com](https://github.com/) → **Sign up** i załóż darmowe konto.

### Krok 2: Znajdź właściwe repozytorium

!!! info "Gdzie zgłosić?"
    Problemy z usługami White Ravens (np. strona nie działa, błąd w konfiguracji) zgłaszaj w repozytorium [whiteravens20/wrservices.link](https://github.com/whiteravens20/wrservices.link).

    Jeśli nie wiesz, gdzie zgłosić — zgłoś właśnie tam, a my przekierujemy do odpowiedniego miejsca.

### Krok 3: Sprawdź, czy ktoś już to zgłosił

1. Wejdź w zakładkę **Issues** w repozytorium.
2. Wpisz słowa kluczowe w wyszukiwarce.
3. Jeśli znajdziesz podobne zgłoszenie — możesz dodać komentarz z dodatkowymi informacjami.

### Krok 4: Utwórz nowe zgłoszenie

1. Kliknij **New Issue**.
2. Wybierz szablon:

    | Szablon | Kiedy użyć |
    |---|---|
    | **Bug Report** | Coś nie działa prawidłowo |
    | **Documentation Issue** | Błąd w dokumentacji |
    | **New Feature / Service Request** | Propozycja nowej funkcji lub usługi |
    | **Blank issue** | Inne — nic z powyższych nie pasuje |

3. Wypełnij formularz — opisz problem tak, jak go widzisz.
4. Kliknij **Submit new issue**.

!!! tip "Język"
    Na GitHub preferujemy język angielski, ale nie jest to wymóg — możesz pisać po polsku.

### Krok 5: Śledź zgłoszenie

Po wysłaniu otrzymasz powiadomienia o komentarzach i zmianach statusu. Jeśli deweloperzy zadadzą pytania — odpowiedz, aby pomóc w rozwiązaniu problemu.

---

## Wskazówki — jak pisać dobre zgłoszenie

- **Opisz, co się stało** — co robiłeś, co powinno się stać, a co stało się zamiast tego
- **Dodaj zrzut ekranu** — obraz mówi więcej niż tysiąc słów
- **Podaj przeglądarkę i system** — np. „Chrome na Windows 11" lub „Firefox na Androidzie"
- **Bądź konkretny** — im więcej szczegółów, tym szybciej problem zostanie naprawiony

??? question "Czy muszę mieć konto GitHub?"
    Nie koniecznie. Możesz też zgłosić problem na naszym Discord — ktoś z zespołu utworzy zgłoszenie za Ciebie.

??? question "Nie wiem, w którym repozytorium zgłosić."
    Zgłoś w [whiteravens20/wrservices.link](https://github.com/whiteravens20/wrservices.link) — przekierujemy do odpowiedniego miejsca.

??? question "Czy mogę zaproponować nową funkcję?"
    Tak! Wybierz szablon **New Feature / Service Request** przy tworzeniu nowego zgłoszenia.
