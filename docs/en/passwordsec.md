# Password security

Passwords are the keys to your online accounts — email, bank, social media. This guide explains how to create strong passwords and manage them in a {==simple and secure==} way.

---

## Why does it matter?

- **One weak password = a threat to many accounts** — if you use the same password everywhere and it leaks from one service, an attacker has access to all your accounts
- **Simple passwords are cracked in seconds** — passwords like `123456`, `qwerty`, or `password123` are the first ones automated programs try
- **Data breaches happen regularly** — even the biggest companies fall victim to attacks

---

## How to create a good password?

### Rules

| Rule | Why? |
|---|---|
| **Minimum 12 characters** | Short passwords are cracked instantly |
| **Upper and lowercase letters, numbers, special characters** | Greater complexity = harder to guess |
| **Different password for each account** | A leak of one password doesn't threaten other accounts |
| **No obvious words** | Avoid names, birth dates, or your favorite band's name |

### Passphrase method

Instead of coming up with complicated strings, use several random words combined together:

!!! example "Examples of strong passwords"
    - `Winter!Kingdom3Tree`
    - `Coffee&Umbrella42Lamp`
    - `Library_Bicycle!Cloud9`

    Such passwords are both **strong** (long, complex) and **easier to remember**.

## Password manager — one password to remember

You don't need to remember dozens of complicated passwords. A **password manager** stores them all in an encrypted vault — you only need to remember one master password.

### How does it work?

1. Install a password manager (app + browser extension).
2. Create one **strong master password** — this is the only one you need to remember.
3. The manager automatically generates and saves strong passwords for each account.
4. When logging in, the manager **automatically fills in the password** — you don't need to know or type it.

!!! tip "White Ravens recommendation"
    We recommend [Bitwarden](https://bitwarden.com/) — a free, open-source password manager, available on computers and phones.

### What to look for when choosing?

- **Encryption** — data should be encrypted with a master password the manufacturer doesn't know
- **Two-factor authentication (2FA)** — additional protection for your vault
- **Availability** — should work on computer, phone, and in the browser
- **Open source** — ability to independently verify security

## Frequently asked questions

??? question "Is a password manager safe? What if someone hacks it?"
    Good managers (like Bitwarden) encrypt all data with your master password. Even if the manufacturer's servers were compromised, the attacker would only get encrypted data that cannot be read without your password.

??? question "Do I have to pay for a password manager?"
    No. Bitwarden offers a free plan that is sufficient for the vast majority of users.

??? question "What if I forget my master password?"
    This is the only password you need to remember. Write it down in a safe place (e.g., on a piece of paper in a safe). Some managers offer a recovery key.

??? question "Is saving passwords in the browser the same?"
    No. Built-in browser managers have much weaker protections. A dedicated password manager offers better encryption, cross-device synchronization, and additional security features.

---

Take care of your security — create a strong, unique password for each account and use a password manager. It's the simplest and most effective way to protect your data online.
