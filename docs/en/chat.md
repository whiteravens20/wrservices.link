# White Ravens Chat

White Ravens Chat is a private instant messenger based on [Element Web](https://element.io/). It works similarly to Messenger or WhatsApp, but with a key difference — all conversations are **end-to-end encrypted** and stored on a private White Ravens server, not with any corporation.

!!! tip "Link"
    [White Ravens Chat](https://chat.wrservices.link/)

**Key features:**

- :material-lock: **Encrypted conversations** — messages can only be read by conversation participants
- :material-devices: **Multiple devices** — use your computer, phone, and tablet simultaneously
- :material-server-security: **Private server** — data stored on White Ravens infrastructure, not with external companies
- :material-open-source-initiative: **Open source** — anyone can check how the application works

---

## How to get started?

### Step 1: Create an account

1. Go to [White Ravens Chat](https://chat.wrservices.link/).
2. Click **Create account**.
3. Fill in the registration form (email address, username, password).
4. Confirm registration.

### Step 2: Log in

- **On computer:** open [chat.wrservices.link](https://chat.wrservices.link/) in your browser and enter your login credentials.
- **On phone/tablet:** download the Element app:

    === "Android"

        Download from [Google Play](https://play.google.com/store/apps/details?id=im.vector.app) or [F-Droid](https://f-droid.org/packages/im.vector.app/).

        When logging in, set the server to: `matrix.wrservices.link`

    === "iOS"

        Download from [App Store](https://apps.apple.com/us/app/element-messenger/id1083446067).

        When logging in, set the server to: `matrix.wrservices.link`

### Step 3: Start a conversation

- Create new chat rooms or join existing ones.
- Send text messages, photos, files, and emoji.
- Organize voice calls and video conferences directly from the app.

## Conversation security

White Ravens Chat protects your conversations by default:

| What is protected | How it works |
|---|---|
| **Message content** | End-to-end encryption — only you and the recipient see the content |
| **Transferred files** | Files are encrypted before sending |
| **Chat history** | Synchronization between devices in encrypted form |

!!! warning "Device verification"
    After logging in on a new device, you'll be asked to **verify** it. This is an important step — it confirms that it's really you. Verify the new device using an existing session (e.g., confirm a phone from your computer).

## Frequently asked questions

??? question "Is White Ravens Chat the same as WhatsApp / Messenger?"
    Functionally, yes — you can send messages, share files, and make voice and video calls. The difference is that your data doesn't go to Meta, Google, or any other corporation — it's stored on a private White Ravens server.

??? question "Can I message people on other Matrix servers?"
    Yes! White Ravens Chat uses the Matrix protocol, which enables communication between different servers — similar to email (someone on Gmail can write to someone on Outlook).

??? question "What happens if I forget my password?"
    Contact the administrator via Discord to reset your password.

??? question "Do I have to pay to use it?"
    No. White Ravens Chat is completely free for community members.
